# ShapeForge³ Latest Professional Upload

Premium static front-end prototype for an AI 3D creation, phone scanning, marketplace and print-ordering platform.

## Upload to GitHub

1. Delete old website files in the repository root.
2. Upload the contents of this ZIP, not the ZIP itself.
3. Make sure `index.html` is in the top/root of the repo.
4. Commit changes.
5. In Netlify, trigger a fresh deploy.

## V6 model vault business model

This version adds a tiered library structure:

- Free Starter: ShapeForge-owned starter files and basic public browsing.
- Subscriber Vault: curated model-source catalogue with licence tracking.
- Pro Originals: premium ShapeForge-owned or partner-approved files.
- Studio: business workflow for commercial licensing, print farm jobs, inventory and analytics.

Important: third-party STL/OBJ/3MF files are not copied into this package unless ShapeForge owns them or has permission. The vault stores source links and licence notes. Only sell or print models commercially when the licence or partner agreement allows it.

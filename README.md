# ShapeForge 3D V2

A cleaner professional version of the ShapeForge 3D website and app shell.

## Main flows
- Create Model
- Scan Object
- Print at Home
- We Print It
- Licensed Model Library
- Pricing
- Admin Dashboard preview

## Deploy
Upload all files to the root of a GitHub repository and deploy to Netlify.
Build command: leave blank.
Publish directory: root / blank.

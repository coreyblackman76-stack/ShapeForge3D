# ShapeForge 3D

Scan. Create. Print. Anywhere.

This is a ready-to-deploy V1 website and installable PWA app for an AI-powered 3D creation and printing platform.

## Included
- Premium landing website
- Installable Android/iPhone PWA shell
- AI Studio interface
- Image/model file upload interface
- Mobile camera scanning workflow
- Print-at-home file workflow
- Print-for-me order workflow
- Quote calculator
- Legal STL/OBJ/3MF model source catalogue
- Creator marketplace screens
- Admin dashboard mockup
- Subscription plans
- Marketing copy blocks
- Offline page and service worker

## Deploy with as few uploads as possible
1. Unzip this package.
2. Upload the folder contents to GitHub.
3. Connect the GitHub repo to Netlify.
4. Publish.
5. Open the site on your phone and choose Add to Home Screen / Install App.

## Important
This V1 contains realistic frontend workflows. Real AI generation, accounts, payment processing, direct printer connection, shipping labels and order database need backend services added next.

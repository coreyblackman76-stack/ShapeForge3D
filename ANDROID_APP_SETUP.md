# ShapeForge 3D Android / iPhone App Setup

## Fastest app path
Deploy the site to Netlify over HTTPS. Open it on Android Chrome or iPhone Safari and install it to the home screen.

## Play Store path later
Use Capacitor or Trusted Web Activity to wrap the PWA as an Android app.

Backend services to add later:
- user accounts
- AI text/image/scan to 3D APIs
- file storage
- payment gateway
- order database
- shipping carrier API
- printer queue integration

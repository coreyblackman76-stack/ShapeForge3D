# ShapeForge 3D Model Library Licence Guide

The app catalogues sources for STL, OBJ and 3MF files. It does not copy or re-host third-party files.

Before commercial use or printing for customers, check each model page for:
- licence type
- commercial printing permission
- attribution requirements
- remix restrictions
- redistribution restrictions

Safest sources for commercial-style workflows are public domain/CC0 models or paid models with clear commercial print licences.

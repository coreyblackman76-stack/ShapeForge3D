# Model Library Licence Guide

ShapeForge³ should catalogue third-party STL/OBJ/3MF models by linking to the original source and storing licence notes. Do not copy, re-host or sell third-party files unless the licence clearly allows it.

## Safe default

- Link to the model source page.
- Show creator name and source platform.
- Show the licence.
- Let users download from the original source unless redistribution is allowed.
- For print-for-me commercial printing, check whether the licence allows commercial use.

## ShapeForge-owned files

Files created by ShapeForge³ can be hosted directly in `assets/models/` and used as free starter printables.


## Commercial use rule for ShapeForge³

Free to download does not automatically mean free to resell. The platform should classify each model as:

- ShapeForge owned: can be sold, bundled, printed and licensed by ShapeForge.
- Partner approved: can be used according to the partner agreement.
- Commercial Creative Commons: may be sold/printed if attribution and licence conditions are met.
- Non-commercial: do not sell prints, paid downloads or include in paid subscription download bundles without permission.
- Unknown: link only, do not monetise.

The Subscriber Vault in this prototype is a catalogue and licensing workflow, not a bulk copied archive.

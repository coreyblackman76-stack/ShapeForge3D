# Model Library Licence Guide

ShapeForge³ should catalogue third-party STL/OBJ/3MF models by linking to the original source and storing licence notes. Do not copy, re-host or sell third-party files unless the licence clearly allows it.

## Safe default

- Link to the model source page.
- Show creator name and source platform.
- Show the licence.
- Let users download from the original source unless redistribution is allowed.
- For print-for-me commercial printing, check whether the licence allows commercial use.

## ShapeForge-owned files

Files created by ShapeForge³ can be hosted directly in `assets/models/` and used as free starter printables.
